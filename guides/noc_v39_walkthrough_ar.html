LLM_Build Prompt — بناء الـ Prompt

v38f-0003 · code
تحضير الـ AI
ما الذي يفعله؟

بيبني system prompt وuser prompt اللي هيتبعتوا للـ LLM. الـ system prompt هو مجموعة قواعد تفصيلية بتقول للـ AI إزاي يفهم أسئلة الـ NOC. الـ user prompt هو سؤال المستخدم ببساطة.

القواعد الأساسية اللي بيعلمها للـ LLM:

لو "كم" + "واقع/down" ← intent = site_down_count، count_only = true
لو "power alarms" أو "باور" ← intent = power_alarm_summary، category = POWER
لو "aging + down" ← intent = down_aging_summary
لو "VIP down" أو "VIP واقع" ← intent = vip_down_sites، isvip = Yes
لو ذُكر اسم منطقة (Alex, Cairo, دلتا…) ← geo_type = region
لو ذُكر كود موقع (ALX3480…) ← geo_type = site
الـ Output
{ chatInput, debugMode, systemPrompt: "You are a telecom NOC classifier...", userPrompt: "User question: ..." }

INT_Operational Guardrails — الحارس مالك v3.9

v38f-0007 · code
شبكة أمان
ما الذي يفعله؟

هذا هو المصدر الوحيد للحقيقة فيما يخص تصحيح الـ intent والـ category. في v3.9، كل منطق التصحيح موجود هنا وبس — مكرر في أي مكان تاني. بيعمل 4 حاجات:

1. Overrides صارمة بناءً على الكلمات المفتاحية (الكلمات تكسب الـ LLM في الأنماط المعروفة)

2. تعبئة الـ intent لو الـ LLM رجع فارغاً أو unknown

3. ضمان التوافق بين intent والـ category

4. تحديد الـ request_profile (simple_query / site_incident_summary / free_chat / placeholder)

الشروط الأساسية (بالأولوية)
Override إجباري: "power" + "aging" في السؤال ← دايماً = power_aging_summary + POWER
Override إجباري: "power" + "parked" ← power_parked_summary + POWER
Override إجباري: "warning"/"critical" + "power" ← power_aging_summary
لو intent فارغ/unknown والسؤال فيه كلمات حوادث ← نشتق الـ intent من الكلمات
← "vip" في السؤال ← vip_down_sites + isvip=Yes
← "tx"/"transport"/"microwave" ← tx_alarm_summary
← كود موقع + "analyze" ← site_incident_summary
← "كم"/"howmany"/"count" ← site_down_count + count_only=true
لو تحية (hi/مرحبا/شكراً) ← free_chat
غير كده مفيش كلمات حوادث ← placeholder
أمثلة
👤 "Power aging summary" الـ override يكسب
intentpower_aging_summary
categoryPOWER
👤 "مرحبا" مسار التحية
modefree_chat
intentfree_chat
👤 "مواقع VIP الواقعة" مطابقة كلمات
intentvip_down_sites
isvipYes
👤 "توقع خسارة الترافيك" ميزة مستقبلية
intentplaceholder_request
modeplaceholder
